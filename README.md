# Agenda de Contatos

Projeto de uma agenda de contatos desenvolvido em linguagem C como trabalho final da disciplina de Algoritmos.

## Funcionalidades

- Adicionar contatos
- Excluir contatos
- Alterar nome ou número
- Listar todos os contatos
- Buscar contatos por nome, ID ou número

## Como executar

É necessário ter um compilador C, como o GCC, instalado.

Clone o repositório:

```bash
git clone https://github.com/Pedro-Helbig/Agenda_de_contatos.git
cd Agenda_de_contatos
```

Compile o programa:

```bash
gcc Listadecontatos.c -o agenda
```

Execute no Windows:

```powershell
.\agenda.exe
```

Execute no Linux ou macOS:

```bash
./agenda
```
